About Me

I’m Asman Malika, a passionate software developer and open-source contributor. I enjoy building backend systems, web applications, and developer tools that are reliable, maintainable, and scalable. Over time, I’ve developed skills in Rust, Go, Python, JavaScript, and web technologies, as well as in data processing and smart contract integration, particularly on platforms like Stellar/Soroban.


I’m driven by solving real-world problems with code and contributing to projects that make technology more accessible and inclusive. I actively participate in open-source communities, enjoy learning from other developers, and love contributing solutions that others can build upon.


I thrive in collaborative environments, value clean and well-documented code, and am always exploring new tools, frameworks, and approaches to improve both myself and the projects I work on.
